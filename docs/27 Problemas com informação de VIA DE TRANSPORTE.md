# Problemas com informação de VIA DE TRANSPORTE

http://comexstat.mdic.gov.br/pt/informativo/27

Distorções de valores em detalhamento de VIA

Com a implantação do novo sistema de exportação, o Portal Único, as sistemáticas de produção de dados estatísticos também precisaram de adaptações. Como o Portal Único ainda está em fase de implantação, alguns processos estão pendentes de conclusão e afetam diretamente a produção e divulgação dos dados. Um dos processos pendentes é o recebimento, por parte do MDIC, das informações de VIA DE TRANSPORTE informado nos registros cadastrados via Portal Único. Esse fornecimento de dados é feito sistematicamente por meio do SERPRO, e atualmente está em andamento a realização de ajustes no serviço de fornecimento para contemplar as informações de VIA DE TRANSPORTE.

Até a completa adaptação do sistema as informações de VIA poderão sofrer distorções. É possível reparar a distorção representada pela VIA DESCONHECIDA. Trata-se na verdade da ausência dessa informação nos registros fornecidos.

Esperemos, o mais breve possível, ajustar por completo o correto processamento das informações de VIA e retomar a divulgação sem distorções. Ressalta-se que as distorções só são percebidas no ano de 2018 para EXPORTAÇÕES. Os anos anteriores (1997-2017) de EXPORTAÇÕES e todo o período para IMPORTAÇÕES permanece com os dados de VIA DE TRANSPORTE sem distorções. Até que isso ocorra, solicitamos não utilizar a informação de VIA DE TRANSPORTE para o período de 2018.

Ressaltamos mais uma vez que todas as divulgações são feitas com base nas informações declaradas de boa fé pelos exportadores e importadores nos sistemas oficiais de exportação e importação.

Dúvidas, críticas, sugestões ou mais detalhamentos podem ser solicitados via Comex Responde.
